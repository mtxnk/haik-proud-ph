# Haik, the Benevolent Sea God - Academic Monograph

**Live Site**: https://haik.proud.ph

## Overview

This repository contains an academic monograph on **Haik**, the benevolent sea god of Tagalog mythology. The work examines Haik as a protective, nurturing ocean deity and contrasts him with other Philippine maritime deities and global ocean gods like Poseidon.

## Content Structure

- **index.html** - Complete academic monograph with scholarly citations
- **Notes section** - Footnoted references to academic sources
- **Bibliography** - External resources and research materials

## Key Topics

1. Haik's role in Tagalog maritime spirituality
2. Comparative analysis with Philippine deities (Amanikable, Maguayan, Bakunawa, Bulan, Sidapa)
3. Contrast with Greek Poseidon
4. Implications for ocean literacy and cultural preservation
